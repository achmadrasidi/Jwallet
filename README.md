## Getting Started

1. Download this Project or you can type `https://github.com/achmadrasidi/jwallet.git`

```sh
$ git clone https://github.com/achmadrasidi/jwallet.git
```

2. Install NPM packages

```sh
$ npm install
```

3. Starting client

```sh
$ npm run dev
```

4. Jwallet Client App is Running
